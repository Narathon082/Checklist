# 📊 ระบบตรวจประเมินคุณภาพข้อมูล (DQA Checklist)
> **สำนักงานปลัดกระทรวงพาณิชย์ (Office of the Permanent Secretary - OPS)**

ระบบเว็บแอปพลิเคชันสำหรับอำนวยความสะดวกในการทำแบบตรวจประเมินคุณภาพข้อมูล (Data Quality Assessment - DQA Checklist) ด้วยตนเอง พัฒนาด้วยโครงสร้าง PHP, JavaScript และระบบฐานข้อมูล MySQL รองรับการใช้งานแบบบันทึกร่างอัตโนมัติ (Autosave) และการพิมพ์รายงานสรุปผลประเมินที่พร้อมพิมพ์ลงกระดาษทันที

---

## 📁 โครงสร้างและการแยกประเภทไฟล์ในระบบ (File Categorization)

เพื่อความเป็นระเบียบและง่ายต่อการดูแลรักษา ระบบได้แบ่งประเภทไฟล์ตามหน้าที่ออกเป็นดังนี้:

### 1. ⚙️ ไฟล์โครงสร้างหลักและการเชื่อมต่อฐานข้อมูล (Core & Database)
*   **[db.php](file:///c:/Checklist/db.php)**: ไฟล์จัดการการเชื่อมต่อ MySQL และมีระบบ **Auto-Migration** ที่ตรวจจับและสร้างตารางฐานข้อมูล (`submissions` และ `agencies`) พร้อมติดตั้งข้อมูลตั้งต้นให้อัตโนมัติเมื่อมีการโหลดใช้งานครั้งแรก
*   **[database.sql](file:///c:/Checklist/database/database.sql)**: ไฟล์ DDL/DML SQL Schema สำหรับนำเข้า (Import) โครงสร้างตารางและข้อมูลรายชื่อหน่วยงานตั้งต้นผ่าน phpMyAdmin

### 2. 🔌 ส่วนควบคุมและรับส่งข้อมูล (API & Logic)
*   **[api.php](file:///c:/Checklist/api.php)**: API Endpoint หลักที่คอยรับส่งข้อมูลฟอร์มในรูปแบบ JSON ระหว่างหน้าเว็บเบราว์เซอร์กับฐานข้อมูล (รองรับการทำ Auto-save และการป้องกัน Session หลุด)
*   **[new.php](file:///c:/Checklist/new.php)**: ล้าง Session เก่าเพื่อสร้างแบบประเมินฉบับใหม่
*   **[edit.php](file:///c:/Checklist/edit.php)**: กำหนด Session ID สำหรับดึงงานประเมินเก่าขึ้นมาแก้ไข แล้วส่งต่อไปยังหน้าฟอร์มหลัก

### 3. 📝 หน้าฟอร์มแบบประเมินทีละขั้นตอน (Multi-page Forms)
*   **[index.php](file:///c:/Checklist/index.php)**: **หน้าที่ 1 & 2** — ส่วนข้อมูลทั่วไปของชุดข้อมูล (ดึงรายชื่อหน่วยงานดำเนินงานจากฐานข้อมูล) และตารางประเมินคุณภาพข้อมูล 5 มิติแรก (ความถูกต้อง, ความสอดคล้อง, ความเป็นปัจจุบัน, ตรงตามต้องการ, ความพร้อมใช้)
*   **[page3.php](file:///c:/Checklist/page3.php)**: **หน้าที่ 3** — ตารางประเมินธรรมาภิบาลข้อมูล (Data Governance)
*   **[page4.php](file:///c:/Checklist/page4.php)**: **หน้าที่ 4** — ตารางประเมินการสอดคล้องนโยบายและมาตรฐานข้อมูล (Alignment)
*   **[page5.php](file:///c:/Checklist/page5.php)**: **หน้าที่ 5** — ส่วนควบคุมและติดตามคุณภาพข้อมูล (Data Quality Monitoring & Control) รวมถึงข้อมูลผู้ประเมิน/ผู้อนุมัติ และวันสิ้นสุดการประเมิน

### 4. 🖥️ หน้าแสดงผลและหน้ารวม (Views & Dashboard)
*   **[dashboard.php](file:///c:/Checklist/dashboard.php)**: หน้ารวมรายการประเมินทั้งหมดในระบบ แสดงสถิติ ค้นหาตามชื่อชุดข้อมูล กรองข้อมูลตามหน่วยงาน รวมถึงมีปุ่มจัดการ (ดูรายงาน, แก้ไข, ลบรายการ)
*   **[view.php](file:///c:/Checklist/view.php)**: หน้าแสดงผลการประเมินฉบับเต็มอย่างละเอียด จัดวางเลย์เอาต์สไตล์ทางการ จัดกลุ่มข้อมูล และซ่อนแถบปุ่มกดอัตโนมัติเมื่อกดสั่งพิมพ์งานออกเป็น PDF/กระดาษ (Print-Friendly)

### 5. 🎨 ไฟล์ทรัพยากรการแสดงผล (Assets & Scripts)
*   **[style.css](file:///c:/Checklist/assets/css/style.css)**: ไฟล์สไตล์ชีต (CSS) หลักที่ใช้ควบคุมความสวยงามของทุกหน้าเว็บ มีการประยุกต์ใช้โทนสีแบรนด์กระทรวงพาณิชย์ (MOC Blue & Gold) และสไตล์แบบเรียบหรูระดับพรีเมียม
*   **[app.js](file:///c:/Checklist/assets/js/app.js)**: ไฟล์ควบคุมการทำงานฝั่งผู้ใช้งาน (Client-side) เช่น การดึง/บันทึกแบบร่างอัตโนมัติไปยังเซิร์ฟเวอร์ทุกครั้งที่มีการพิมพ์ (Autosave every 5s / change), การบันทึกแบบร่างสำรองลงเบราว์เซอร์ (Local Storage) กรณีเซิร์ฟเวอร์ขัดข้อง, และการจัดการปุ่มนำทางย้อนกลับ-ถัดไป
*   **[ops-logo.jpg](file:///c:/Checklist/assets/images/ops-logo.jpg)**: โลโก้ของสำนักงานปลัดกระทรวงพาณิชย์ (OPS)

## 🛠️ ขั้นตอนการติดตั้งและเปิดใช้งาน (Installation Guide)

### 1. การเตรียมความพร้อมทางซอฟต์แวร์
*   ติดตั้งโปรแกรมจำลองเซิร์ฟเวอร์ เช่น **XAMPP** (ดาวน์โหลดได้ที่ [apachefriends.org](https://www.apachefriends.org))
*   เปิดใช้งานโมดูล **Apache** และ **MySQL** ใน XAMPP Control Panel

### 2. การคัดลอกไฟล์โครงการ
*   สร้างโฟลเดอร์ชื่อ `Checklist` ไว้ใต้โฟลเดอร์จัดเก็บเว็บของ XAMPP:
    `C:\xampp\htdocs\Checklist`
*   คัดลอกไฟล์ทั้งหมดในโปรเจกต์นี้ไปวางในโฟลเดอร์ดังกล่าว

### 3. การเตรียมฐานข้อมูล (ทำได้ 2 วิธี)
*   **วิธีที่ 1 (อัตโนมัติ):** เมื่อติดตั้งเสร็จและเรียกใช้งานหน้าเว็บผ่านเบราว์เซอร์ครั้งแรก ระบบจะทำการสร้างฐานข้อมูลชื่อ `checklist` และโครงสร้างตารางทั้งหมดรวมถึงรายชื่อหน่วยงาน 12 แห่งให้คุณโดยอัตโนมัติทันที
*   **วิธีที่ 2 (ทำด้วยตนเองผ่าน phpMyAdmin):**
    1. เข้าไปที่เบราว์เซอร์ไปที่ลิงก์ `http://localhost/phpmyadmin/`
    2. สร้างฐานข้อมูลใหม่ชื่อ `checklist` (Collation: `utf8mb4_unicode_ci`)
    3. กดเลือกฐานข้อมูล `checklist` -> กดแท็บ **Import** -> อัปโหลดไฟล์ `database.sql` จากโปรเจกต์ -> กดปุ่ม **Go**

## ✨ คุณสมบัติเด่นของระบบ (Key Features)
*   **Autosave & Local Draft Backup**: ระบบบันทึกร่างข้อมูลลงฐานข้อมูลโดยอัตโนมัติเพื่อป้องกันข้อมูลหาย พร้อมกลไกสำรองข้อมูลลงเบราว์เซอร์หากการเชื่อมต่อมีปัญหา
*   **Dynamic Agencies**: รายชื่อหน่วยงานในดรอปดาวน์ถูกเชื่อมโยงกับฐานข้อมูลโดยตรง สามารถเข้าไป เพิ่ม/ลด/แก้ไข รายชื่อหน่วยงานได้ทันทีผ่านตาราง `agencies` ใน phpMyAdmin โดยไม่ต้องแก้โค้ด
*   **Official Print Design**: หน้ารายงานสรุปผลประเมิน (`view.php`) ออกแบบโครงสร้างรองรับการพิมพ์อย่างสวยงาม มีการจัดแบ่งหน้าตารางอย่างสมบูรณ์แบบ
*   **Secure Record Deletion**: สามารถกดลบรายการตรวจประเมินได้จากหน้าแดชบอร์ด โดยมีระบบกล่องยืนยันการลบแบบปลอดภัย
